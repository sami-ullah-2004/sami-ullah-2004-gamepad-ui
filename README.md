<p align="justify">
A modern and interactive virtual gamepad interface built using HTML, CSS, and JavaScript.
This project displays a sleek on-screen controller with a responsive design, smooth button interactions, and a clean gaming-style layout.

<h1 align="center">Features</h1>

.Modern & clean UI design

.Fully responsive (Mobile + Desktop)

.A, B, X, Y functional buttons

.D-Pad directional controls

.Smooth animations & transitions

.Lightweight and fast
